# Shopify-Mirakl Integration

This integration automatically syncs:
- Products: Shopify → Mirakl
- Inventory: Shopify → Mirakl
- Orders: Mirakl → Shopify
- Tracking: Shopify → Mirakl

## Setup Instructions

See the full setup guide in our chat conversation.

## Environment Variables Required

- SHOPIFY_STORE_NAME
- SHOPIFY_ACCESS_TOKEN
- MIRAKL_API_URL
- MIRAKL_API_KEY
- MIRAKL_SHOP_ID
